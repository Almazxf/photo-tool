# RenPy Photo Tool

Программа для пакетной обработки фото под RenPy: ресайз, обрезка/letterbox/растяжение,
конвертация JPG/PNG, отзеркаливание, прогресс-бар. Работает как portable .exe —
без установки, без следов на компьютере.

## Структура проекта

```
renpy-photo-tool/
├── go.mod
├── main.go              -- окно приложения (Fyne GUI)
├── imaging.go            -- логика обработки картинок
└── .github/workflows/build.yml   -- автосборка .exe на GitHub
```

## Шаг 1. Создать репозиторий на GitHub

1. Зайди на https://github.com/new
2. Имя репозитория, например: `renpy-photo-tool`
3. Сделай его **Public** (для Public-репозиториев GitHub Actions бесплатны без лимитов)
4. НЕ добавляй README, .gitignore, лицензию — оставь пустым
5. Нажми "Create repository"

GitHub покажет тебе адрес репозитория вида:
`https://github.com/ТВОЙ_ЛОГИН/renpy-photo-tool.git`

## Шаг 2. Залить файлы проекта

Распакуй присланный архив в любую папку на компьютере, открой там терминал
(в Windows: правый клик в папке → "Открыть в терминале" или через `cmd`/PowerShell)
и выполни по очереди:

```bash
git init
git add .
git commit -m "Первая версия RenPy Photo Tool"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЛОГИН/renpy-photo-tool.git
git push -u origin main
```

Если попросит логин/пароль — GitHub давно не принимает обычный пароль, нужен
**Personal Access Token** (его можно создать в Settings → Developer settings →
Personal access tokens → Generate new token, права достаточно `repo`).
Вместо пароля при пуше вставляешь этот токен.

## Шаг 3. Дождаться сборки

1. На странице репозитория открой вкладку **Actions**
2. Увидишь запущенный workflow "Build Windows EXE" — он соберётся примерно 2-4 минуты
3. Если всё прошло зелёным ✅ — переходи в **Releases** (справа на главной странице репо)
   и скачивай `RenPyPhotoTool.exe`
4. Если что-то покраснело ❌ — открой лог упавшего шага, скопируй текст ошибки
   и пришли мне, я поправлю код

## Шаг 4. Использование

1. Скачай `RenPyPhotoTool.exe` на флешку
2. Запусти прямо с флешки (двойной клик) — ничего не устанавливается
3. В открывшемся окне:
   - перетащи фото или целую папку с фото прямо в окно программы
   - либо нажми "Выбрать фото или папку..."
   - настрой разрешение, обрезку, формат, отзеркаливание, качество
   - нажми "Начать обработку"
4. Готовые файлы появятся в подпапке `output` рядом с .exe
5. После работы можно просто вынуть флешку — на компьютере не останется
   ни записей в реестре, ни файлов установки, ни истории браузера (программа
   браузер вообще не использует)

## Если нужно что-то поменять

Просто пришли мне, что изменить — поправлю `main.go` / `imaging.go`, ты заменишь
файлы в своей папке, сделаешь `git add . && git commit -m "fix" && git push`,
и GitHub Actions автоматически пересоберёт новую версию .exe.
